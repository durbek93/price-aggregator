# Price Aggregator (Сводный прайс)

Скрипт собирает данные из Excel-прайсов в папке `prices` и формирует сводную матрицу по препаратам производителя Материа Медика.

## Установка

1. Python 3.8+
2. Создать виртуальное окружение и установить зависимости:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

Дважды щёлкните по `run_svodny.bat` или выполните в папке проекта:

```bash
.\.venv\Scripts\python.exe svodny_price.py --config config.example.yml --inputs prices --producers "Материа Медика" "Материя Медика" "Materia Medica" "Материа Медика Холдинг" --producer-matrix
```

Результат: файл в формате `output\Сводный прайс + ДД.ММ.ГГГГ.xlsx` — матрица препаратов (строки) и дистрибьюторов (столбцы).

## Подсчёт ассортимента по прайсам

Подсчитывает количество **уникальных товаров** в каждом прайсе из папки `prices` и сохраняет таблицу в `output`.

```bash
.\.venv\Scripts\python.exe assortment_prices.py --config config.example.yml --inputs prices
```

Результат: файл в формате `output\Ассортимент товаров (ДД.ММ.ГГГГ).xlsx`.

## Подсветка минимальной цены в сводных файлах

Считывает сводные прайсы из папки `output`, находит минимальную цену по каждому препарату и подсвечивает соответствующие ячейки зелёным цветом в самих файлах.

Запуск:

```bash
.\.venv\Scripts\python.exe cheapest_svodny_prices.py --inputs output
```

или двойным кликом по `run_cheapest_svodny.bat`.

Результат: изменённые файлы `output\Сводный прайс*.xlsx` (ячейки с минимальными ценами выделены зелёным).

## Настройка

- **config.example.yml** — кандидаты названий колонок (наименование, цена, производитель и т.д.) для разных форматов прайсов.
- В коде можно расширять каталог препаратов и синонимы в `build_catalog_materia_medica()` и список дистрибьюторов в `friendly_source_label()`.
